1.Зачем нужен GIT и какие задачи решают с его помощью?

-  инструмент для управления версиями

2.Что такое репозиторий GIT и где он находится?

 - репозиторий находится на GIT bush. Там все файлы версии которых находятся под контролем. 
репозиторий можно создать из любой папки на компьютере (локальный) 
и привязать и выгрузить на гит (удаленный)

3.Для чего используется файл .gitignore?

- для указания в нем файлов которые должны быть проигнорированны

4.Что такое "коммит"?

- контрольная точка состояния проекта на текущий момент времени


5.Укажите базовые команды в рамках работы с GIT:

git init
git add
git commit
git status
git push
git branch
git rm -rf
git remote add

Настройка аккаунта локального пользователя. 
git config
git config --global user.name
git config --global user.email

Создание репозитория.
git init

Добавление файлов в индекс и выполнение коммита.
git add .
git commit -m 'текст'

Просмотр журнала коммитов.
git log
